### Prerequisites

In order to run this project you need to:

```sh
Set up Webpack
linters and jest.
```

### Setup

Clone this repository to your desired folder:

```sh
  cd To-do-list-project
  git clone https://github.com/HERMON-1995/My-movies-capstone-project.git
```
### Install
 
 ```sh
 cd To-do-list-project
 ```
 ```sh
 npm install
 ```
